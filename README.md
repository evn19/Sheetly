# ExcelChat
A Windows desktop messaging application packaged as a single EXE.

## Concept
- Messaging app UI - mimics ios26 imessage liquid glass design
- Messages stored in Excel (.xlsx) files
- Excel files are shared via OneDrive
- Each chat (group or P2P) has its own Excel file
- Each message is a new row:
  Timestamp | Sender | Message ect

## Constraints
- No external services
- No dependencies except excel
- Excel is the backend
